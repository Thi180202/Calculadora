# Calculadora
# Projeto - Calculadora em Python

## Descrição
Este projeto consiste em uma calculadora desenvolvida em linguagem Python, executada via script shell no Linux.

---

## Explicação do código Python

O script em Python realiza operações matemáticas básicas:

- Soma
- Subtração
- Multiplicação
- Divisão

O usuário insere os valores e escolhe a operação desejada, e o programa retorna o resultado.

---

## Tecnologias utilizadas

- Python
- Shell Script (.sh)
- Linux (Ubuntu)
